项目经历

MuseFlow AI - AI 驱动的智能音乐播放平台

Vue 3 TypeScript Vite Pinia Vue Router Express SSE Sass HTML5 Audio DashScope / OpenAI Compatible API

项目描述：

基于 Vue 3 + TypeScript 构建的智能音乐播放平台，整合搜索、歌单、MV、评论、播放队列、收藏/本地音乐、最近播放同步与 AI 自然语言控播能力；通过 Node 中间层聚合音乐数据接口与大模型接口，形成从音乐发现、播放控制到用户音乐资产沉淀的完整闭环。

项目亮点：

  • 流式响应体验优化：基于 SSE 实现 AI 回复逐段输出与中断控制，用户无需等待完整结果返回即可看到响应内容，减少 AI 交互过程中的等待空白感，提升对话实时性。
  • 路由懒加载与页面拆包：对首页、搜索、歌单、MV、设置、迷你播放器等业务页面进行路由级懒加载，减少首屏一次性加载资源；生产构建中多数业务页 JS 控制在约 4KB - 17KB，主布局 gzip 后约 19.5KB。
  • 稳定播放链路建设：封装歌曲音源解析、音源过期缓存、代理流式播放、Range 请求转发、进度恢复、音量/播放模式持久化等能力，解决外链音源失效、刷新丢进度、切歌状态错乱等播放器核心体验问题。
  • 搜索与内容消费体验优化：搜索页支持歌曲/歌单/MV 分类、分页、URL 状态同步和过期请求丢弃，并可一键将当前页可播放歌曲转为播放队列；同时补齐评论、MV 播放、歌手/专辑跳转等链路，提升从发现歌曲到播放消费的转化效率。
  • 播放状态持久化节流：对进度条拖动、音量调整等高频操作进行节流保存，降低频繁写入 localStorage 带来的性能损耗，同时保留刷新后恢复播放进度、音量和播放模式的能力。
